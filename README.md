## ✨ 核心功能

- **积分商城**: 观众可通过每日签到、开通大航海、赠送礼物等方式获取积分，并用于兑换虚拟道具或实体礼品。支持后台管理、上下架商品、积分变更记录等。

- **直播间打卡签到**: 观众每天可签到获取积分，并记录累计/连续签到天数，让直播间形成健康的活跃循环。

- **PK 播报**: PK 对战开启前，自动播报对手的活跃数据和贡献榜信息，支持自定义播报模板，让直播节奏更带感。

- **礼物答谢**: 收到礼物后自动播报感谢语，可设置金额门槛并自定义多条个性化文案随机展示。

- **定时广告**: 定时推送预设内容，支持多条文案随机发送，用于宣传加群、抽奖、活动、商单等。

- **进房欢迎**: 观众进入直播间时自动欢迎，支持多套话术随机展示，提升互动体验。

- **感谢关注**: 当用户关注直播间时自动感谢，并支持多条话术。

- **感谢分享**: 用户分享直播间时自动感谢，支持多条文案的随机展示。

- **自动回复**: 当弹幕触发预设关键词时自动回复，支持自定义多套回复方案，并随机输出，让机器人更自然不生硬。

- **自动禁言**: 与自动回复联动，对违规触发内容的观众进行临时禁言；并支持通过赠送指定金额的礼物自动解除禁言。

- **数据分析**: 记录直播间弹幕与礼物信息，生成用户每日消费统计、盲盒盈亏数据、弹幕词云等分析报告，帮助主播了解观众互动和贡献情况，更好地优化直播策略。
